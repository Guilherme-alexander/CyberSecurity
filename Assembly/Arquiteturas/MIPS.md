# MIPS

## Introdução

MIPS é uma arquitetura RISC bastante utilizada em pesquisas e sistemas embarcados.

Foi muito popular entre os anos 1990 e 2000.

---

# Características

- RISC
- Pipeline simples
- Fácil aprendizado

---

# Onde foi usada

- PlayStation 1
- Nintendo 64
- Roteadores
- Equipamentos embarcados

---

# Registradores

```
$0
$at
$v0
$a0
$t0
$s0
$gp
$sp
$fp
$ra
```

---

# Exemplo

```asm
li $t0,10
addi $t0,$t0,5
```

---

# Curiosidades

Muitos cursos de arquitetura de computadores utilizam MIPS para ensinar Assembly.
