# ARM64 (AArch64)

## Introdução

ARM64 é a evolução da arquitetura ARM para 64 bits.

Também chamada de **AArch64**.

---

# Características

- 64 bits
- RISC
- Alto desempenho
- Muito eficiente energeticamente

---

# Registradores

```
X0-X30
```

Versão de 32 bits:

```
W0-W30
```

---

# Sistemas

- Android
- iOS
- macOS Apple Silicon
- Linux

---

# Exemplo

```asm
MOV X0,#100
ADD X0,X0,#50
```

---

# Curiosidades

Todos os processadores Apple Silicon utilizam ARM64.
