# RISC-V

## Introdução

RISC-V é uma arquitetura aberta criada pela Universidade da Califórnia, Berkeley.

Seu diferencial é ser totalmente livre de royalties.

---

# Características

- Open Source
- Modular
- RISC
- Escalável

---

# Registradores

```
x0-x31
```

Também possuem apelidos:

```
zero
ra
sp
gp
tp
a0
a1
```

# Curiosidades

RISC-V é considerada por muitos como a principal concorrente aberta das arquiteturas x86 e ARM, sendo adotada em projetos acadêmicos, dispositivos embarcados e, cada vez mais, em processadores comerciais.
