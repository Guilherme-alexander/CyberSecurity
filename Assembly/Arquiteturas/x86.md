# x86 (IA-32)

## Introdução

A arquitetura **x86** é uma família de processadores de 32 bits desenvolvida pela Intel. Seu nome surgiu a partir da sequência de processadores:

- 8086
- 80186
- 80286
- 80386
- 80486

Com o lançamento do Intel 80386, a arquitetura passou a suportar 32 bits e ficou conhecida como **IA-32**.

---

# Características

- 32 bits
- Registradores de 32 bits
- Endereçamento de até 4 GB
- Compatível com versões anteriores
- Muito utilizada entre 1995 e 2010

---

# Registradores Principais

| Registrador | Função |
|-------------|---------|
| EAX | Acumulador |
| EBX | Base |
| ECX | Contador |
| EDX | Dados |
| ESI | Source Index |
| EDI | Destination Index |
| EBP | Base Pointer |
| ESP | Stack Pointer |
| EIP | Instruction Pointer |

---

# Sistemas Operacionais

- Windows XP

- BSD

Algumas convenções comuns:

---

# Exemplo

```asm
mov eax, 10
add eax, 5
```

Resultado:

```
EAX = 15
```

---

# Vantagens

- Grande compatibilidade
- Muito material de estudo
- Simples para iniciantes

---

# Limitações

- Apenas 4 GB de memória
- Menos registradores que x64
- Performance inferior

---

# Curiosidades

O x86 ainda é encontrado em sistemas legados, caixas eletrônicos, equipamentos industriais e softwares antigos.
