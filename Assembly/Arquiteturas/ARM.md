# ARM

## Introdução

ARM é uma arquitetura RISC criada na década de 1980.

Seu foco é baixo consumo de energia.

É muito utilizada em dispositivos embarcados.

---

# Características

- RISC
- 32 bits
- Baixo consumo
- Poucas instruções
- Alta eficiência

---

# Onde é usada

- Smartphones antigos
- Raspberry Pi
- Microcontroladores
- IoT

---

# Registradores

```
R0-R15
```

Exemplo:

```
R13 = SP
R14 = LR
R15 = PC
```

---

# Exemplo

```asm
MOV R0,#10
ADD R0,R0,#5
```

---

# Vantagens

- Econômica
- Simples
- Muito eficiente

---

# Curiosidades

ARM significa originalmente **Acorn RISC Machine**.
